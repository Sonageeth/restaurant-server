                        

                        Restaurant Backend Creation using JSON-SERVER
                       - -----------------------------------------------

               1. Create a new folder for Backend - restaurant-SERVER
               2. Create package.json file - npm init
               3. Install json-server - npm i json-server(node_modules,package-lock.json kitti)
               4. Create index.js file - (Apply code backend creation) 
               5. Run Server - nodemon index.js
